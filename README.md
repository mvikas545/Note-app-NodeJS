# Note-app-NodeJS
A app to keep your all notes using Node JS

# Allowed operations
1. Add new note
2. Remove a note
3. Read a note
4. List all notes

### To add a new note
node app.js add --title="note title" --body="note description"

### To remove a note
node app.js remove --title="note title"

### To read a note
node app.js read --title="note title"

### To get list of notes
node app.js list
